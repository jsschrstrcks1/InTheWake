## Rule
Every HTML page MUST include a `<meta name="ai-summary" content="...">` tag with non-empty content. This is the first-line AI-consumption summary of the page.

## Why (rationale)
ICP-Lite / ICP-2 protocol. The summary tells AI systems (and the site's internal generators) what the page is about in a single bounded string. Without it, AI consumption degrades to scraping. The port validator hard-errors on absence (rule `ai_summary_missing`, line 664-668).

## Pass example
```html
<meta name="ai-summary" content="Practical cruise-planning guide for Nassau, Bahamas. Walk from Prince George Wharf to downtown in 5 minutes; Cable Beach by jitney ($1.25, 25 min); Atlantis taxi ($8 one-way)."/>
```

## Fail example
```html
<!-- missing meta tag entirely -->
<meta name="description" content="Nassau cruise port guide"/>
```
The `description` meta is not a substitute. ICP-001 requires the `ai-summary` name specifically.

## Fix guidance
Add the tag to `<head>`. The content should be a standalone sentence (or short paragraph) — see ICP-002 for the length / standalone-sentence rule that governs the content itself.
