## Rule
Every HTML page must declare `<!doctype html>` (or `<!DOCTYPE html>`) as the first tag after the Soli Deo Gloria invocation comment. The 2025-11 audit found 62 pages missing it. Both the YAML standards and post-write-validate.sh enforce this.

## Why (rationale)
Without DOCTYPE, browsers render in quirks mode — inconsistent box model, different default styles, unpredictable layout behavior. Quirks mode is a silent regression that can go unnoticed for weeks until someone tests in a different browser.

## Fix guidance
Add `<!doctype html>` immediately after the closing `-->` of the Soli Deo Gloria comment. It must be the first HTML tag; nothing else comes between the comment and the doctype.
