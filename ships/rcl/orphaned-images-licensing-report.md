## Licensing Information Guide

### Wikimedia Commons Images
For images from Wikimedia Commons, licensing information can be found at their respective Commons pages. Common licenses for ship photography include:

- **CC BY-SA 2.0, 3.0, 4.0** (Creative Commons Attribution-ShareAlike)
  - Requires attribution to author
  - Derivatives must use same license
  - Commercial use allowed

- **CC BY 2.0, 3.0, 4.0** (Creative Commons Attribution)
  - Requires attribution to author
  - Derivatives can use different license
  - Commercial use allowed

- **Public Domain** / **CC0**
  - No attribution required
  - Can use for any purpose

### Standard Attribution Format

For Wikimedia Commons images, use this attribution format:

```
[Image Title] by [Author Name], [License Type], via Wikimedia Commons
Source: [Commons URL]
```

Example:
```
"Brillance of the Seas passing Aurland" by [Author], CC BY-SA 2.0, via Wikimedia Commons
Source: https://commons.wikimedia.org/wiki/File:Brillance_of_the_Seas_passing_Aurland_(17413746643).jpg
```

### FOM Images
Images with "FOM-" prefix likely come from a specific photography collection (possibly "Friends of Madeira" or similar). License information for these images requires:
1. Identifying the source organization
2. Contacting them for licensing terms
3. Obtaining permission if needed

2. **Manual Image Analysis** (9 images)
   - Open each low-confidence image
   - Look for ship names on hull, funnel, or signage
   - Cross-reference with itinerary data if dates/locations available

3. **Wikimedia Commons License Extraction** (52 images)
   - Visit each Commons URL listed above
   - Extract specific license type (CC BY-SA, CC BY, etc.)
   - Document author names
   - Note any special attribution requirements

4. **Oslo Series Investigation** (Images #2, #5-7)
   - Check photographer RalfR's Commons portfolio
   - Look for additional metadata or descriptions
   - Cross-reference with Oslo port records for September 2, 2014

5. **Multi-Ship Filename** (Image #67)
   - Open image to visually identify which ship
   - Look for distinguishing features (size, design elements)

### Ship Identification Tips

When manually reviewing images, look for:
- **Ship name on hull** (usually near bow)
- **Funnel design** (RCL's distinctive blue funnel with rock climber logo)
- **Size class** (helps narrow down possibilities)
- **Distinctive features** (rock wall, AquaTheater, Ultimate Abyss, etc.)
- **Launch year** (if date in filename, limits possible ships)

## Appendix A: Wikimedia Commons URL Construction

For Wikimedia Commons images, URLs follow this pattern:
```
https://commons.wikimedia.org/wiki/File:{filename}.{extension}
```

**Note:** Replace `.webp` extensions with the original format (usually `.jpg` or `.png`). Wikimedia Commons stores originals, and `.webp` is often a converted format.
