## Rule
Every page must have a "Skip to main content" link as the first focusable element, visually hidden until focused, that jumps to the page's main landmark (`#main` or `<main id="main-content">`).

## Why (rationale)
WCAG 2.1 AA SC 2.4.1 (Bypass Blocks). Keyboard and screen-reader users should not have to tab through 30 nav links on every page. The site serves disabled travelers — this is non-optional.

## Pass example
```html
<body>
  <a href="#main" class="skip-link">Skip to main content</a>
  <header>...</header>
  <nav>...</nav>
  <main id="main">...</main>
</body>
```

## Fail example
```html
<body>
  <header>...</header>  <!-- skip-link absent -->
  <nav>...</nav>
  <main>...</main>
</body>
```
Validator emits `missing_skip_link: Missing skip link (recommended for accessibility)` at warn severity.

## Fix guidance
Add a visually-hidden anchor as the first body child. CSS: `.skip-link { position: absolute; top: -40px; left: 0; } .skip-link:focus { top: 0; }`. Target must match the main element's id.
