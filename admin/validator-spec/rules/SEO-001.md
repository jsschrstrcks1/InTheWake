## Rule
Every page should include four Open Graph meta tags: `og:title`, `og:description`, `og:image`, `og:url`. These control how the page appears when shared on Facebook, LinkedIn, iMessage, Slack, Discord, and most messaging platforms. No validator currently enforces — orphan.

## Why (rationale)
A page shared on Facebook without OG tags renders as a bare URL with no preview. With them, readers see a title, description, and image — dramatically higher click-through. The site's social-sharing surface is free marketing; missing OG tags waste it.

## Fix guidance
Add the four tags in `<head>`. `og:title` = page title, `og:description` = ai-summary content (or description meta), `og:image` = absolute URL to hero image, `og:url` = canonical URL. Use absolute URLs for og:image (relative URLs break on most platforms).
