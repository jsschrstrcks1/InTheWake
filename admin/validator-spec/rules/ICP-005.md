## Rule
Every page must include `<meta name="last-reviewed" content="YYYY-MM-DD"/>`. This tag declares when the content was last human-reviewed. It is required by all three validators (port, ship, venue).

## Why (rationale)
Cruise information rots fast. Port prices change, ships get redeployed, restaurants close. The `last-reviewed` meta is a commitment that a human looked at this content on that date and vouched for it. Without it, AI consumers and human readers have no way to know if the content is from 2023 or last week.

ICP-005 is the presence check. ICP-006 handles the format validation, ICP-007 handles staleness.

## Pass example
```html
<meta name="last-reviewed" content="2026-04-15"/>
```

## Fail example
```html
<!-- tag absent -->
```
Validator emits: `last-reviewed meta tag is missing`.

## Fix guidance
Add the tag. Use today's date only if you actually reviewed the page today — fabricating a current date is a dishonesty that will eventually be caught (and is exactly the kind of clever shortcut the `careful-not-clever` guardrail exists to prevent).
