## Rule
The primary site navigation across every page must contain the 12 canonical top-level and child links in the order specified by `NAVIGATION_STANDARDS_ADDENDUM_v3.008.md`. "Cruise Lines" in particular is never removable from its parent. A validator today does not enforce the canonical ordering or the full child-link set; this rule is on the orphan list awaiting implementation.

## Why (rationale)
Nav consistency is a silent trust signal — readers learn where things live and expect the same shape on every page. When sessions add/remove/reorder nav items ad-hoc, users become disoriented. Standards document this; no validator currently checks it. That's exactly the S-only gap this rule records.

## Pass example
```html
<nav aria-label="Primary">
  <ul>
    <li><a href="/">Home</a></li>
    <li><a href="/planning.html">Planning</a>
      <ul>
        <li><a href="/first-cruise.html">First Cruise</a></li>
        <li><a href="/packing-lists.html">Packing</a></li>
        <li><a href="/disability-at-sea.html">Disability at Sea</a></li>
        <li><a href="/solo.html">Solo Cruising</a></li>
        <li><a href="/drink-packages.html">Drink Packages</a></li>
        <li><a href="/internet-at-sea.html">Internet at Sea</a></li>
      </ul>
    </li>
    ... etc per NAVIGATION_STANDARDS_ADDENDUM
  </ul>
</nav>
```

## Fail example
```html
<nav aria-label="Primary">
  <ul>
    <li><a href="/">Home</a></li>
    <li><a href="/planning.html">Planning</a></li>
    <li><a href="/ships.html">Ships</a></li>
    <!-- "Cruise Lines" removed; order compressed; children missing -->
  </ul>
</nav>
```
No validator flags this today. A reader clicking through multiple pages finds nav shape shifting — a reliability failure invisible to automation.

## Fix guidance
Until a validator enforces this, it's a manual review item. When building or revising a page, copy the nav block from a known-good page (index.html is gold standard) rather than hand-rolling. A Phase 2+ validator can implement this check by: (a) hashing the nav structure across pages and alerting on divergence, or (b) asserting presence of each canonical href.
