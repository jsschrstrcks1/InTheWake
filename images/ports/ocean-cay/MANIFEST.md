## Before/after verification protocol for the crop/straighten edits

For each photo where edits are applied:

1. Save the original to `images/ports/ocean-cay/originals/<filename>` before touching it.
2. Apply the rotation/crop per the spec above.
3. View both side-by-side. Confirm:
   - the horizon is true (water meets sky in a straight line within ±0.2°),
   - the subject still has appropriate breathing room (not clipped),
   - no important detail was cropped out (e.g. the rogue apostrophe on the menu board, the swim-zone buoys in #2),
   - dimensions are reasonable for web (long edge ≤ 2400px, file size ≤ 500KB at quality 82),
   - alt text in the page still accurately describes the cropped result.
4. If the edit made the photo worse — restore the original and skip the edit for that photo. The before/after comparison is the gate; "I applied my recommended crop" is not by itself success.
