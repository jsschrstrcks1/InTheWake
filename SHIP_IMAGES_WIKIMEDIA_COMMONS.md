## Image Selection Guidelines

**Preferred image types (in order of priority):**
1. **Aerial/from above** - Shows full ship profile
2. **In port/docked** - Shows ship at port terminal
3. **Next to another Royal Caribbean ship** - Comparison shots

**Image quality requirements:**
- Minimum 1920px wide (will be converted to WebP)
- Quality 85% for WebP conversion
- Clear, well-lit exterior shots
- Avoid interior-only photos

**Attribution requirements:**
Each image needs:
- Author/photographer name
- License type (CC BY, CC BY-SA, etc.)
- Source link to Wiki Commons page
- Proper `<figcaption>` in HTML

**Example attribution HTML:**
```html
<figcaption class="tiny">
  Photo by [Author] via
  <a href="[Wiki Commons URL]" target="_blank" rel="noopener">Wikimedia Commons</a>
  ([CC BY-SA 3.0])
</figcaption>
```

## Download Instructions

1. Visit the Wikimedia Commons category page for each ship
2. Select 3-4 best exterior/port/aerial images
3. Download full resolution versions
4. Save to: `/assets/ships/[wikimedia-commons-slug]/`
5. Convert to WebP using: `python3 convert_to_webp.py <input_image>`
6. Note attribution details for HTML integration
