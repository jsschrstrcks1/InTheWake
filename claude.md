## Key Principles

1. **Soli Deo Gloria** — Every HTML file MUST have the invocation before line 20
2. **ITW-Lite** — AI-first → Human-first → Google second
3. **ICP-Lite v1.4** — Required metadata on all pages
4. **Last-reviewed dates** — Update `last-reviewed` meta AND `dateModified` JSON-LD on every edit
5. **Careful, not clever** — Read before editing, verify, document
