## Rule
Prose must have natural sentence-length variance. Two failure modes: (1) all sentences roughly the same length (monotone machine tell), (2) strict complex-then-simple alternation (false variation — still mechanical). Natural prose has clusters: a run of short declaratives followed by a longer landing sentence, then a mid-length transition, then another short burst.

## Fix guidance
Read the paragraph aloud. If every sentence takes the same breath, compress some and expand others. The Like-a-human cadence guide describes specific patterns: "compression and release", "shrinking sentences at emotional climax", "pause-and-pivot after intense passages."
