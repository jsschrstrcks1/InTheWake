## Wikimedia Commons Images

### Template Format
When adding Wikimedia Commons images, use this format:

```
#### [Image Title]
- **File**: `path/to/image.jpg`
- **Source**: [Wikimedia Commons URL]
- **Author**: [Photographer/Creator Name]
- **License**: [CC BY-SA 4.0 / CC0 / etc.]
- **Changes**: [None / Cropped / Resized / etc.]
- **Attribution**: Photo by [Author] / [Source], [License]
```

### Example
```
#### Royal Caribbean Harmony of the Seas
- **File**: `assets/images/ships/harmony-hero.jpg`
- **Source**: https://commons.wikimedia.org/wiki/File:Harmony_of_the_Seas_example.jpg
- **Author**: John Photographer
- **License**: CC BY-SA 4.0
- **Changes**: Cropped to 16:9 aspect ratio
- **Attribution**: Photo by John Photographer / Wikimedia Commons, CC BY-SA 4.0
```
